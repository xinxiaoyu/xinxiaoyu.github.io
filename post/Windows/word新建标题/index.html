Word新建标题

2011年7月5日
| Windows
| 阅读

  1. 视图 –> 导航窗格，显示标题

  2. 在标题行回车后会出现什么(如2.2 test，调整大纲级别为2级后)


   2.2 test    | 2.2 test      | 2.2 test          | 2.2 test
   -------------------------------------------------------------
   enter       | enter+tab     | enter+enter       | shift+enter
   -------------------------------------------------------------
   2.3 test    | 2.2.1 test    | 3 test(level 2)   | so what? 
       |                                                |         
       v                                                v
   2.3 test，出现后，选中-->右击-->段落-->大纲级别-->正文文本，写内容
