Rabbitmq_权限问题

2020年7月23日
| Error RabbitMQ
| 阅读

发布消息时报如下错误：

pika.exceptions.ChannelClosedByBroker: (403, “ACCESS_REFUSED - access to queue ‘hello’ in vhost ‘/’ refused for user ‘admin’

这是权限问题，需要赋予相应用户的 vhost ‘/‘权限。

1

rabbitmqctl set_permissions -p / admin ".*" ".*" ".*"

注意引号，我因为打成中文的引号所以出错了。
